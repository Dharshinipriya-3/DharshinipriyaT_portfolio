Disaster Risk Prediction & Monitoring System

A full-stack flood-risk system that logs rainfall and water-level readings, classifies each reading as Low, Medium, or High risk using a trained decision-tree model, and keeps a searchable history of past records for review.

FlaskSQLitescikit-learnDecision Tree ClassifierPandasJoblib
  • Backend built in Flask, with a dedicated API blueprint and separate routes for the live dashboard and a past-records page.
  • A SQLite database (disaster.db) persists every reading — rainfall, water level, computed risk level, and timestamp — for later lookup.
  • A decision-tree model is trained on rainfall and water-level pairs and exported with joblib, so predictions run instantly at request time instead of being recomputed from scratch.
  • Risk thresholds are tuned in three tiers, so the same reading always maps to a consistent, explainable Low / Medium / High output.
